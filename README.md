# encriptador-alura
Oracle ONE Challenge 1, encriptador de código
Proyecto del programa Oracle ONE en el cual proponen crear un encriptador de texto, al reemplazar este las vocales de una palabra por un conjunto de otras letras, encriptando de esta forma la palabra, y de igual modo, desencriptando el código cuando se le ingresa el texto encriptado. Para esto se utiliza un archivo HTML y CSS para generar la interfaz del usuario y un archivo con código Javascript con las funciones para encriptar y desencriptar.
Proyecto eleborado en el mes de marzo del año 2024 por Christiam Arroyave.

INSTRUCCIONES

1. Ingresa un texto sin tildes o mayusculas en la zona "Ingresa un texto para codificar".
2. Hacer clic en el botón "CODIFICAR".
3. En el lado derecho de las pantallas de computadoras o tabletas aparecerá su texto encriptado.
4. Se activará un botón "COPIAR" en la parte inferior del texto encriptado para pasarlo nuevamente a la zona de encriptación.
5. Puede codificar nuevamente su texto, generando una encriptación más compleja y dificil de identificar.
6. Puede descodificar o descifrar el texto tantas veces como sea necesario para lograr un mensaje legible.
7. Disfruta enviando mensajes encriptados por sus redes sociales favoritas.
